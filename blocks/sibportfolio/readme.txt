Для установки портфолио требуется создать папку с файлами портфолио sibportfolio в moodle/blocks. Также можно воспользоваться стандартными средствами Moodle по установке плагинов типа "блок".

В папке pix могут храниться картинки для категорий файлов портфолио. Название картинки является идентификатором категории. Например, категории с id равным 3 соответствует картинка с именем 3.png. Используется только формат png.
